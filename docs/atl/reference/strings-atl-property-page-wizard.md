# <a name="strings-atl-property-page-wizard"></a>문자열, ATL 속성 페이지 마법사
속성 페이지와 관련 된 텍스트를 제공 합니다.  
  
 **제목**  
 속성 페이지의 탭에 표시 되는 텍스트를 설정 합니다.  
  
 **문서 문자열**  
 페이지를 설명 하는 텍스트 문자열을 설정 합니다. 이 문자열은 속성 시트 대화 상자에서 표시할 수 있습니다. 속성 프레임이 상태 줄 또는 도구 설명에 설명을 사용할 수 있습니다. 현재 표준 속성 프레임은이 문자열을 사용 하지 않습니다.  
  
 **도움말 파일**  
 속성 페이지를 사용 하는 방법을 설명 하는 도움말 파일의 이름을 설정 합니다. 이 이름은 경로가 포함 되지 않습니다. 사용자가 누르면 **도움말**, 프레임 속성 페이지 레지스트리 항목의 CLSID 아래에 있는 HelpDir 키의 값에 명명 된 디렉터리에서 도움말 파일을 엽니다.
