# <a name="viewing-resource-symbols"></a>리소스 기호 보기
### <a name="to-view-resource-symbols"></a>리소스 기호를 보려면  
  
1.  [리소스 뷰](../windows/resource-view-window.md)를.rc 파일을 마우스 오른쪽 단추로 클릭 합니다.  
  
    > [!NOTE]
    >  프로젝트에 .rc 파일이 아직 없는 경우 [새 리소스 스크립트 파일 만들기](../windows/how-to-create-a-resource-script-file.md)를 참조하세요.  
  
2.  선택 **리소스 기호** 의 리소스 기호 테이블을 보려면 바로 가기 메뉴에서는 [리소스 기호 대화 상자](../windows/resource-symbols-dialog-box.md)합니다.  
  
    > [!NOTE]
    >  미리 정의 된 기호를 보려면 확인는 **읽기 전용 기호 표시** 확인란 합니다.  
  

  
 **Requirements**  
  
 Win32
