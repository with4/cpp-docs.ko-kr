# <a name="editing-properties-for-an-activex-control"></a>ActiveX 컨트롤 속성 편집
외부 공급 업체에서 제공 하는 ActiveX 컨트롤 속성 및 특성이 있을 수 있습니다. ActiveX 컨트롤에 대 한 속성은 속성 창에 표시 됩니다. ActiveX 컨트롤의 작성자가 만든 모든 속성 페이지에 표시 됩니다는 또한는 **속성 페이지** 대화 상자 (특정 ActiveX 컨트롤에 대 한 속성 페이지를 보려면는 에서속성페이지단추를클릭[속성 창](/visualstudio/ide/reference/properties-window)).  
  
 다양 한 탭 ActiveX 컨트롤의 일부로 제공 되는 속성 시트에 따라 ActiveX 컨트롤에 대 한 속성 페이지에 표시 됩니다.  
  
> [!NOTE]
>  속성 페이지를 사용 하 여 ActiveX 컨트롤을 편집 하는 다음 절차가 적용 됩니다. 탐색 하 고 새 속성 창에서 ActiveX 속성을 편집할 수도 있습니다.  
  
### <a name="to-edit-properties-for-an-activex-control"></a>ActiveX 컨트롤에 대 한 속성을 편집 하려면  
  
1.  선택 된 **ActiveX** 제어 합니다.  
  
2.  에 **보기** 메뉴를 클릭 하 여 **속성 페이지** 속성을 확인 합니다.  
  
3.  속성 페이지에서 필요에 따라 변경 내용을 확인 합니다.  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](https://msdn.microsoft.com/library/f45fce5x.aspx) 에 *.NET Framework 개발자 가이드입니다.* 수동으로 관리 되는 프로젝트에 리소스 파일을 추가, 리소스 액세스, 정적 리소스 표시 및 속성에 리소스 문자열 할당에 대 한 정보를 참조 하십시오. [데스크톱 앱에 대 한 리소스 파일 만들기](https://msdn.microsoft.com/library/xbx3z216.aspx)합니다. 전역화 및 지역화의 관리 되는 응용 프로그램의 리소스에 대 한 정보를 참조 하십시오. [전역화 및 지역화.NET Framework 응용 프로그램](https://msdn.microsoft.com/library/h6270d0z.aspx)합니다.
