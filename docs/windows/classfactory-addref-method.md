# <a name="classfactoryaddref-method"></a>ClassFactory::AddRef 메서드
현재 ClassFactory 개체에 대 한 참조 횟수를 증가 시킵니다.  
  
## <a name="syntax"></a>구문  
  
```  
STDMETHOD_(  
   ULONG,  
   AddRef  
)();  
```  
  
## <a name="return-value"></a>반환 값  
 성공하면 S_OK이고, 그렇지 않으면 실패를 설명하는 HRESULT가 발생합니다.

 **헤더:** module.h  
  
 **네임스페이스:** Microsoft::WRL
