# <a name="hstringreferenceget-method"></a>HStringReference::Get 메서드
기본 HSTRING 핸들의 값을 검색 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HSTRING Get() const throw()  
```  
  
## <a name="return-value"></a>반환 값  
 기본 HSTRING 핸들의 값입니다.  
  
## <a name="requirements"></a>요구 사항

 **Namespace:** Microsoft::WRL::Wrappers
