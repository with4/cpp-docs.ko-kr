# <a name="deleting-an-entry-from-an-accelerator-table"></a>액셀러레이터 키 테이블에서 항목 삭제
### <a name="to-delete-an-entry-from-an-accelerator-table"></a>액셀러레이터 키 테이블에서 항목을 삭제하려면  
  
1.  액셀러레이터 키 테이블에서 해당 아이콘을 두 번 클릭 하 여 열고 [리소스 뷰](../windows/resource-view-window.md)합니다.  
  
    > [!NOTE]
    >  프로젝트에 .rc 파일이 아직 없는 경우 [새 리소스 스크립트 파일 만들기](../windows/how-to-create-a-resource-script-file.md)를 참조하세요.  
  
2.  삭제하려는 항목을 선택합니다. (키를 누른 채는 **CTRL** 또는 **SHIFT** 키를 클릭 하는 동안 여러 항목을 선택 합니다.)  
  
3.  마우스 오른쪽 단추로 클릭 하 고 선택 **삭제** 바로 가기 메뉴에서 (또는 선택 **삭제** 에서 **편집** 메뉴).  
  
 \- 또는 -  
  
-   Delete 키를 누릅니다.  
  
 관리 되는 프로젝트에 리소스를 추가 정보를 참조 하십시오 [데스크톱 응용 프로그램의 리소스](/dotnet/framework/resources/index) 에 *.NET Framework 개발자 가이드입니다.*
