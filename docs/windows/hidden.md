# <a name="hidden"></a>hidden
항목이 존재 하지만 하지 사용자 기반 브라우저에 표시할지 나타냅니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
[hidden]  
  
```  
  
## <a name="remarks"></a>설명  
 **숨겨진** c + + 특성에 동일한 기능을는 [숨겨진](http://msdn.microsoft.com/library/windows/desktop/aa366861) MIDL 특성입니다.
