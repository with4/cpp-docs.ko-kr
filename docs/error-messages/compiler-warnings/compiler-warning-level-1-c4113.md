# <a name="compiler-warning-level-1-c4113"></a>컴파일러 경고 (수준 1) C4113
'identifier1' 'identifier2' 매개 변수 목록이 다릅니다.  
  
 함수 포인터는 다른 함수 포인터에 할당 된 있지만 함수 형식 매개 변수 목록이 일치 하지 않습니다. 할당은 수정 없이 컴파일됩니다.
