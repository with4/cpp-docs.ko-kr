# <a name="compiler-warning-level-1-c4811"></a>컴파일러 경고(수준 1) C4811
pragma conform(forScope, show) 값 == value  
  
 사용 하는 경우이 경고는 발생는 **표시** 의 옵션은 [준수](../../preprocessor/conform.md) pragma입니다. *value* 가 현재의 확인 값입니다.
