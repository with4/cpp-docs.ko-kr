# <a name="compiler-warning-level-1-c4685"></a>컴파일러 경고(수준 1) C4685
템플릿 매개 변수를 분석하는 경우 '> >'가 있어야 하는데 '>>'가 왔습니다.  
  
 템플릿 정의가 올바르게 종료되지 않았습니다.
