# <a name="compiler-warning-level-1-c4088"></a>컴파일러 경고 (수준 1) C4088
'function': 실제 매개 변수 'number', 'number' 형식 매개 변수에서 포인터가 일치 하지 않습니다.  
  
 해당 형식 및 실제 매개 변수는 한 서로 다른 수준의 간접 참조를 갖습니다. 실제 매개 변수가 변경 되지 않은 상태로 전달 됩니다. 호출된 된 함수 포인터와 해당 값을 해석합니다.
