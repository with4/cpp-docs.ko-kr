# <a name="compiler-warning-level-1-c4143"></a>컴파일러 경고(수준 1) C4143
pragma 'same_seg'가 지원되지 않으므로 __based 할당을 사용하세요.  
  
 **#pragma same_seg** 는 더 이상 지원되지 않습니다. 대신 [__based](../../cpp/based-pointers-cpp.md) 키워드를 사용합니다.
