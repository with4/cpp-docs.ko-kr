 지정한 대상에 대 한 명령 블록의 명령 0이 아닌 종료 코드를 반환 했습니다. /K 옵션 NMAKE 빌드 관련 되지 않은 부분을 계속 처리 하 NMAKE 세션이 끝날 때 종료 코드 1 지시 합니다.  
  
 NMAKE 경고 이면 지정된 된 대상 자체를 다른 대상에는 종속 [U4011](../../error-messages/tool-errors/nmake-warning-u4011.md) 이 경고 다음 합니다.
