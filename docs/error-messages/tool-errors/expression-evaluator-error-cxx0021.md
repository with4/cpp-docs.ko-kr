# <a name="expression-evaluator-error-cxx0021"></a>식 계산기 오류 CXX0021
구조체 또는 스칼라로 사용 된 공용 구조체  
  
 식에서 구조체 또는 공용 구조체를 사용 했지만 요소가 지정 되었습니다.  
  
 구조체 또는 공용 구조체 변수를 조작할 때는 변수 이름을 필드 한정자 없이 단독으로 나타날 수 있습니다. 식에서 구조체 또는 공용 구조체를 사용 하면 원하는 특정 요소와 정규화 되어야 합니다.  
  
 값을 식에 사용 되는 요소를 지정 합니다.  
  
 이 오류는 can0021과 동일 합니다.
