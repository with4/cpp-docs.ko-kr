# <a name="resource-compiler-error-rc2167"></a>리소스 컴파일러 오류 RC2167
인식할 수 없는 VERSIONINFO 필드입니다. 시작 또는 쉼표가 필요 합니다.  
  
 인식할 수 없는 필드를 찾을 수는 **FIXED** 의 일부로 **VERSIONINFO** 구조체 선언 합니다. A **VERSIONINFO** 필드는 쉼표로 구분 하 여 DWORD 여야 합니다.
