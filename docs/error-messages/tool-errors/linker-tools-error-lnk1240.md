# <a name="linker-tools-error-lnk1240"></a>링커 도구 오류 LNK1240
IDL 내용을 컴파일하지 못했습니다.  
  
 링커 MIDL을 생성 컴파일 포함된 IDL 있지만 문제가 발생 했습니다. MIDL에 의해 지정 된 오류를 확인 합니다.
