# <a name="resource-compiler-error-rc2169"></a>리소스 컴파일러 오류 RC2169
리소스 파일의 파일 이름이 2.03 형식이 아닙니다.  
  
 지정된 된 리소스는 버전 2.03 이전의 형식을 사용 합니다. 리소스 파일 변환 해야 하거나 버전 3.00 이상의 형식을 사용 하는 다시 생성 합니다.
