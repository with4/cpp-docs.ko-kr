# <a name="nmake-fatal-error-u1007"></a>NMAKE 심각한 오류 U1007
큰따옴표 이름에 사용할 수 없습니다  
  
 지정한 대상 이름이 나 파일 이름이 포함 된 큰따옴표 (**"**).  
  
 큰따옴표는 파일 이름이 둘러쌀 수 있지만 그 안에 포함 될 수 없습니다.
