# <a name="linker-tools-error-lnk1143"></a>링커 도구 오류 LNK1143
잘못 되었거나 손상 된 파일: COMDAT 섹션 번호에 대 한 기호가 없습니다  
  
 개체 파일이 손상 된 경우에이 오류가 발생할 수 있습니다.  
  
 파일을 다시 빌드해야 합니다.
