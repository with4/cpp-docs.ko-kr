# <a name="nmake-fatal-error-u1052"></a>NMAKE 심각한 오류 U1052
찾을 수 없습니다 ' filename' 파일  
  
 NMAKE로 다음 중 하나를 지정한 파일을 찾을 수 없습니다.  
  
-   **/F** 옵션  
  
-   **! 포함** 전처리 지시문  
  
-   At 기호 (**@**)에 대 한 지시 파일 지정자  
  
 파일이 있으며 파일 이름 철자가 올바른지 확인 합니다.
