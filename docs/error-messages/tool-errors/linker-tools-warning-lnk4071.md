# <a name="linker-tools-warning-lnk4071"></a>링커 도구 경고 LNK4071
다음 링크에 대해 증분 링크할 수 없습니다.  
  
 링크에는 하나 이상의 기호에 대 한 정의가 여러 개 찾을 수 있지만 [/force](../../build/reference/force-force-file-output.md) 또는 **/FORCE:MULTIPLE** 오류에 관계 없이 출력 파일을 만드는 데 사용 합니다. 링크 (.ilk) 증분 상태 파일을 삭제 합니다.
