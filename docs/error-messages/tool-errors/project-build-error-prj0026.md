# <a name="project-build-error-prj0026"></a>프로젝트 빌드 오류 PRJ0026
'File' 지시 파일에 사용자의 ANSI 코드 페이지로 변환할 수 없는 유니코드 내용이 있습니다.  
  
 ***파일의 내용을 유니코드***  
  
 프로젝트 시스템에 사용자의 현재 ANSI 코드 페이지로 제대로 변환 될 수 없는 지시 파일에 유니코드 내용을 발견 했습니다.  
  
 이 오류에 대 한 확인을 ANSI 사용 하거나 코드 페이지를 컴퓨터에 설치 하 고 시스템 기본값으로 설정 하는 응답 파일의 내용을 업데이트 하기 위해서입니다.
