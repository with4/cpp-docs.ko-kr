# <a name="expression-evaluator-error-cxx0048"></a>식 계산기 오류 CXX0048
호출 시퀀스가 지원 되지 않습니다  
  
 식에 지정 된 함수는 C 식 계산기에서 지원 되지 호출 시퀀스를 사용 합니다. 조사식 창에서에서이 함수를 호출할 수 없습니다.  
  
 이 오류는 can0048과 동일 합니다.
