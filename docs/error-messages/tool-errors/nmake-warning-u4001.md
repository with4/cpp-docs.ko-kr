# <a name="nmake-warning-u4001"></a>NMAKE 경고 U4001
명령줄 에서만에서 명령 파일을 호출할 수 있습니다.  
  
 호출 되는 명령 파일을는 at 기호 (**@**) 지정자는 다른 명령 파일에 대 한 사양을 포함할 수 없습니다. 이러한 중첩 허용 되지 않습니다. 사양을 무시 되었습니다.
