# <a name="linker-tools-warning-lnk4220"></a>링커 도구 경고 LNK4220
잘못 된 '링커 옵션' 값은 ''; 기본값이 선택 됩니다.  
  
 범위를 벗어난 값을 지정 했습니다는 [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) 옵션입니다. 에 대 한 기본값 **/TLBID** 는 1입니다.
