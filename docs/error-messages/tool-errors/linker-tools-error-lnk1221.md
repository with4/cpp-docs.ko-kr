# <a name="linker-tools-error-lnk1221"></a>링커 도구 오류 LNK1221
하위 시스템을 유추할 수 없으므로 정의 해야 합니다.  
  
 링커는 하위 시스템을 유추할 대상으로 지정할 충분 한 정보는 없습니다.  
  
 이 오류를 해결 하려면 사용 하 여 [/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)합니다.
