# <a name="linker-tools-warning-lnk4229"></a>링커 도구 경고 LNK4229

 지시문을 통해 전달 되는 [주석](../../preprocessor/comment-c-cpp.md) pragma 올바르지 않습니다. 링커 무시 `/directive`합니다.
