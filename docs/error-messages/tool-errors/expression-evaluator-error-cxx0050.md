# <a name="expression-evaluator-error-cxx0050"></a>식 계산기 오류 CXX0050
왼쪽:: 클래스/구조체/공용 구조체 이어야 합니다  
  
 범위 결정 연산자의 왼쪽에 있는 기호 (`::`) 클래스, 구조체 또는 공용 구조체 없습니다.  
  
 이 오류는 can0050과 동일 합니다.
