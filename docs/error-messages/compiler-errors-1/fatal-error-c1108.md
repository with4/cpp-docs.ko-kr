# <a name="fatal-error-c1108"></a>심각한 오류 C1108
'dll' DLL을 찾을 수 없습니다.  
  
 경로에서 지정된 DLL(***dll***)을 찾을 수 없습니다.  
  
 이 오류를 해결하려면 Visual C++를 다시 설치하거나, 설치 CD에서 해당 .dll 파일을 컴퓨터로 복사합니다.
