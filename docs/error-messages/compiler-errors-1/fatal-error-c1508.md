# <a name="fatal-error-c1508"></a>심각한 오류 C1508
컴파일러 한계: 'function': 인수가 65,535바이트가 넘습니다.  
  
 함수에 대한 정식 매개 변수가 65535바이트 제한을 초과했습니다.
