# <a name="compiler-error-c2222"></a>컴파일러 오류 C2222
예기치 않은 형식 'type': 기본 클래스 또는 멤버가 필요합니다.  
  
 이니셜라이저 목록은 기본 클래스 또는 형식의 멤버만 초기화할 수 있습니다. 이 오류를 해결하려면 기본 클래스 또는 해당 형식의 멤버만 이니셜라이저 목록에서 초기화되는지 확인합니다.
