# <a name="compiler-error-c2219"></a>컴파일러 오류 C2219
구문 오류: 형식 한정자는 이후 여야 합니다. ' *'  
  
 형식 한정자 (`const` 또는 `volatile`) 허용 되지 않은 위치를 표시 합니다.
