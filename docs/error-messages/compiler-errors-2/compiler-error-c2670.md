# 컴파일러 오류 C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 함수 템플릿이 number번째 매개 변수를 'type' 형식으로부터 변환할 수 없습니다.  
  
 매개 변수를 필수 형식으로 변환하지 못했습니다.  
  
 명시적 변환을 만들면 이 오류를 해결할 수 있습니다.
