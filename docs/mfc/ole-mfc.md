# <a name="ole-mfc"></a>OLE(MFC)
프로그램에서 OLE 기능을 구현 합니다. 여러 가지 방법으로 사용자 인터페이스를 영향을 줍니다.  
  
-   비주얼 편집 (내부 활성화) 프로그램의 창에 다른 프로그램의 사용자 인터페이스를 표시 하 고 다른 프로그램에서 항목으로 프로그램의 메뉴를 수정 합니다.  
  
-   끌어서 놓기는 개체 내에서 windows 및 여러 프로그램 간에 끌 수 있습니다.  
  
-   추적기 비주얼 편집 하는 동안 개체의 상태를 시각적 표시를 제공 하 고 끌어서 놓습니다.
