# <a name="ole-classes"></a>OLE 클래스
OLE 클래스 프로그램 ActiveX의 사용자에 게 제공 하는 쉬운 방법을 제공 ActiveX API에 쉽게 액세스할 수는 다른 응용 프로그램 프레임 워크 클래스를 사용 합니다. ActiveX를 사용 하 여 다음을 수행할 수 있습니다.  
  
-   사용자가을 만들고 텍스트, 그래픽, 사운드, 스프레드시트를 포함 하 여 여러 응용 프로그램에서 생성 된 데이터를 포함 하는 문서를 편집할 수 있는 복합 문서 또는 다른 종류의 데이터를 만듭니다.  
  
-   복합 문서에 포함 될 수 있는 OLE 개체를 만듭니다.  
  
-   OLE 끌어서 놓기를 사용 하 여 응용 프로그램 간에 데이터를 복사 합니다.  
  
-   자동화를 사용 하 여 다른 한 프로그램을 제어할 수 있습니다.  
  
-   ActiveX 컨트롤 및 ActiveX 컨트롤 컨테이너 (이전의 OLE 컨트롤 및 OLE 컨트롤 컨테이너 각각)를 만듭니다.
