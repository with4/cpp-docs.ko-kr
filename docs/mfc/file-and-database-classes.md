# <a name="file-and-database-classes"></a>파일 및 데이터베이스 클래스
이 클래스를 사용 하는 데이터베이스 또는 디스크 파일에 정보를 저장할 수 있습니다. 데이터베이스 클래스의 세 가지 집합이 있습니다-OLE DB, ODBC 및 DAO-유사한 기능을 제공 합니다. OLE DB 그룹 OLE DB를 사용 하 여 구현 되 고 작동 하는 OLE DB 소비자 템플릿, DAO 그룹 데이터 액세스 개체를 사용 하 여 구현 됩니다 및 ODBC 그룹 Open Database Connectivity를 사용 하 여 구현 됩니다. 표준 파일, 활성 스트림 및 HTML 스트림을 조작 하기 위한 클래스 집합도 있습니다.
