# <a name="writing-mfc-applications"></a>MFC 응용 프로그램 작성
이 문서에서는 응용 프로그램을 개발 하는 초기 단계에 설명 합니다. 첫째, 작성 하는 응용 프로그램의 종류를 결정 해야 합니다. 여러 선택 항목에서 논의 했습니다 [응용 프로그램 디자인 선택](../mfc/application-design-choices.md)합니다. 응용 프로그램 설정 됩니다.  
  
-   인터넷 또는 인트라넷에서 실행  
  
-   클라이언트 또는 서버에서 실행  
  
-   브라우저에서 또는 독립 실행형 응용 프로그램으로 실행  
  
-   COM 또는 활성 기술을 사용 하 여  
  
-   WinInet 또는 비동기 모니커를 사용 하 여 데이터를 다운로드 합니다.  
  
 결정 하는 데 어떤 클래스는 응용 프로그램에 대 한 적절 한 결정 합니다. 도 도움을 응용 프로그램을 구성 하려면 응용 프로그램 마법사를 실행할 때 선택 항목을 결정 합니다.  
  
 인터넷 응용 프로그램에 대 한 초기 디자인 관련 결정 사항을 변경한 후 시작 하려면 응용 프로그램 마법사를 사용할 수 있습니다. 응용 프로그램 마법사를 사용 하 여 기본 응용 프로그램을 만들고 다음 문서에 설명 된 대로 코드를 수정 하려면:
