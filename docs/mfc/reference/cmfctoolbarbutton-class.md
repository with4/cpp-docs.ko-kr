## <a name="remarks"></a>설명  
 `CMFCToolbarButton` 개체는 도구 모음에 있는 컨트롤입니다. 해당 동작은 일반 단추를 유사 합니다. 이 개체에 이미지 및 텍스트 레이블을 할당할 수 있습니다. 도구 모음 단추의 명령 id입니다. 수도 있습니다. 사용자가 도구 모음 단추를 클릭 하면 프레임 워크는이 ID를 지정 하는 명령을 실행 합니다.  
  
 일반적으로 도구 모음 단추 사용자 지정할 수 있습니다: 사용자 수 간에 도구 모음에서 단추를 끌어 및 복사, 붙여넣기, 삭제 및 텍스트 레이블 및 이미지를 편집 합니다. 사용자 도구 모음 사용자 지정을 방지 하려면 두 가지 방법 중 하나에서 도구 모음을 잠글 수 있습니다. 설정 하거나 합니다 `bLocked` 플래그를 호출 하는 경우 TRUE [CMFCToolBar::LoadToolBar](../../mfc/reference/cmfctoolbar-class.md#loadtoolbar), 개별 단추의 명령 ID를 사용 하 여 보호 된 명령의 전체 목록에 추가 또는 [CMFCToolBarButton:: SetProtectedCommands](#setprotectedcommands) 메서드.  
  
 `CMFCToolBarButton` 응용 프로그램에서 도구 모음 이미지의 전역 컬렉션에서 이미지를 표시 하는 개체입니다. 이러한 컬렉션은 부모 도구 모음에서 유지 관리 [CMFCToolBar 클래스](../../mfc/reference/cmfctoolbar-class.md)합니다. 자세한 내용은 [CMFCToolBarImages 클래스](../../mfc/reference/cmfctoolbarimages-class.md)합니다.  
  
 사용자가 도구 모음 단추를 클릭 하면 해당 부모 도구 모음 마우스 메시지를 처리 하 고 단추에 적절 한 조치를 통신 합니다. 단추에 유효한 명령 ID가 부모 도구 모음을 부모 프레임 WM_COMMAND 메시지를 보냅니다.  
  
 `CMFCToolBarButton` 클래스는 다른 도구 모음 단추 클래스에 대 한 기본 클래스와 같은 [CMFCToolBarMenuButton 클래스](../../mfc/reference/cmfctoolbarmenubutton-class.md)하십시오 [CMFCToolBarEditBoxButton 클래스](../../mfc/reference/cmfctoolbareditboxbutton-class.md), 및 [ CMFCToolBarComboBoxButton 클래스](../../mfc/reference/cmfctoolbarcomboboxbutton-class.md)합니다.

##  <a name="show"></a>  CMFCToolBarButton::Show  
 표시 하거나 단추를 숨깁니다.  
  
```  
void Show(BOOL bShow);
```  
  
### <a name="parameters"></a>매개 변수  
 [in] *bShow*  
 단추를 표시할지 여부를 지정 하는 부울 값입니다. 이 매개 변수가 TRUE 인 경우 단추 표시 됩니다. 매개 변수가 FALSE 인 경우 단추가 숨겨집니다.  
  
### <a name="remarks"></a>설명  
 프레임 워크가 해당 부모 도구 모음 크기를 조정할 때 도구 모음 단추의 표시 여부를 업데이트 하려면이 메서드를 호출 합니다. 이 메서드를 호출 하는 프레임 워크 *bShow* 단추 도구 모음에서 범위 내에서 더 이상 적합 한 경우 FALSE로 설정 합니다. 이 메서드를 호출 하는 프레임 워크 *bShow* 도구 모음의 범위 내에 맞도록 다시 단추 크기 조정 후 경우 TRUE로 설정 합니다.  
  
 사용 된 [CMFCToolBarButton::SetVisible](#setvisible) 일반 단추의 표시 여부를 설정 하는 방법입니다.  
  
 이 메서드를 호출 합니다 [CMFCToolBarButton::OnShow](#onshow) 메서드 버튼의 표시 여부 상태를 업데이트 한 후입니다.
