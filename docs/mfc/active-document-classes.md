# <a name="active-document-classes"></a>액티브 문서 클래스
Internet Explorer 5.5 등의 웹 브라우저의 전체 클라이언트 창에서 또는 활성 문서를 지 원하는 Microsoft Office Binder 같은 활성 컨테이너에 활성 문서를 표시할 수 있습니다.  
  
 [CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)  
 액티브 문서 인터페이스 및를 초기화를 매핑하고 액티브 문서 개체를 활성화 합니다.  
  
 [CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)  
 OLE 서버 동사를 구현 액티브 문서 서버에 특정 한 합니다.  
  
 [COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)  
 액티브 문서 포함을 구현합니다.
