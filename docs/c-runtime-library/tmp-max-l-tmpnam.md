# <a name="tmpmax-ltmpnam"></a>TMP_MAX, L_tmpnam
## <a name="syntax"></a>구문  
  
```  
#include <stdio.h>  
```  
  
## <a name="remarks"></a>설명  
 `TMP_MAX`는 `tmpnam` 함수가 생성할 수 있는 고유한 파일 이름의 최대 수입니다. `L_tmpnam`은 `tmpnam`에 의해 생성되는 임시 파일 이름의 길이입니다.
