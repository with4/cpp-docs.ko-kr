# <a name="rmtmp"></a>_rmtmp

임시 파일을 제거합니다.

## <a name="syntax"></a>구문

```C

int _rmtmp( void );
```

## <a name="return-value"></a>반환 값

**_rmtmp** 닫히고 삭제 된 임시 파일의 수를 반환 합니다.

## <a name="remarks"></a>설명

**_rmtmp** 함수는 현재 디렉터리에 있는 모든 임시 파일을 정리 합니다. 함수에서 만든 파일에만 제거 **tmpfile**; 임시 파일이 생성 된 동일한 디렉터리에만 사용 합니다.
