# <a name="strninc-wcsninc-mbsninc-mbsnincl"></a>_strninc, _wcsninc, _mbsninc, _mbsninc_l

이동 하 여 문자열 포인터 **n** 문자입니다.

> [!IMPORTANT]
> **_mbsninc** 및 **_mbsninc_l** 는 Windows 런타임에서 실행 되는 응용 프로그램에서 사용할 수 없습니다. 자세한 내용은 [유니버설 Windows 플랫폼 앱에서 지원되지 않는 CRT 함수](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md)를 참조하세요.

## <a name="syntax"></a>구문

```C
char *_strninc(
   const char *str,
   size_t count
);
wchar_t *_wcsninc(
   const wchar_t *str,
   size_t count
);
unsigned char *_mbsninc(
   const unsigned char *str,
   size_t count
);
unsigned char *_mbsninc(
   const unsigned char *str,
   size_t count,
   _locale_t locale
);
```

### <a name="parameters"></a>매개 변수

*str*<br/>
소스 문자열입니다.

*count*<br/>
문자열 포인터를 증분하는 문자 수입니다.

*locale*<br/>
사용할 로캘입니다.

## <a name="return-value"></a>반환 값

에 대 한 포인터를 반환 하는 이러한 각 루틴 *str* 후 *str* 증가 *count* 문자 또는 **NULL** 경우 제공 된 포인터는 **NULL**합니다. 경우 *count* 에 있는 문자의 수를 보다 크거나 *str*, 결과가 정의 되지 않습니다.

## <a name="remarks"></a>설명

**_mbsninc** 증가 함수 *str* 여 *count* 멀티 바이트 문자입니다. **_mbsninc** 에 따라 멀티 바이트 문자 시퀀스를 인식 된 [멀티 바이트 코드 페이지](../../c-runtime-library/code-pages.md) 에서 현재 사용 합니다.

### <a name="generic-text-routine-mappings"></a>제네릭 텍스트 라우팅 매핑

|Tchar.h 루틴|_UNICODE 및 _MBCS 정의되지 않음|_MBCS 정의됨|_UNICODE 정의됨|
|---------------------|--------------------------------------|--------------------|-----------------------|
|**_tcsninc**|**_strninc**|**_mbsninc**|**_wcsninc**|

**_strninc** 및 **_wcsninc** 단일 바이트 문자 문자열 및 버전의 와이드 문자 문자열 **_mbsninc**합니다. **_wcsninc** 및 **_strninc** 이러한 매핑을 위해서만 제공 되 고 그렇지 않으면 사용할 수 없습니다. 자세한 내용은 [일반 텍스트 매핑 사용](../../c-runtime-library/using-generic-text-mappings.md) 및 [일반 텍스트 매핑](../../c-runtime-library/generic-text-mappings.md)을 참조하세요.

**_mbsninc_l** 대신 전달 된 로캘 매개 변수를 사용 한다는 점을 제외 하 고는 동일 합니다. 자세한 내용은 [Locale](../../c-runtime-library/locale.md)을 참조하세요.
