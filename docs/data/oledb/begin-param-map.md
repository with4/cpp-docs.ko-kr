# <a name="beginparammap"></a>BEGIN_PARAM_MAP
매개 변수가 맵 항목의 시작을 표시 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
BEGIN_PARAM_MAP(  
x  
 )  
  
```  
  
#### <a name="parameters"></a>매개 변수  
 *x*  
 [in] 사용자 레코드 클래스의 이름입니다.  
  
## <a name="remarks"></a>설명  
 매개 변수를 사용 하 여 [명령을](https://msdn.microsoft.com/en-us/library/ms724608.aspx)합니다.  
  
## <a name="example"></a>예  
 예를 참조는 [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md) 매크로입니다.
