# <a name="endp"></a>ENDP
프로시저의 끝을 표시 *이름* 이전에 시작 된 **PROC**합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
name  
 ENDP  
  
```  
  
## <a name="remarks"></a>설명  
 참조 [PROC](../../assembler/masm/proc.md)합니다.
