# <a name="operator-mod"></a>연산자 MOD
나눌 때 나머지 (모듈로)의 정수 값을 반환 *expression1* 여 *expression2*합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
expression1 MOD expression2  
```
