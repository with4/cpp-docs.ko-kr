# <a name="operator-imagerel"></a>연산자 IMAGEREL
이미지 상대 오프셋을 반환 `expression`합니다.  
  
## <a name="syntax"></a>구문  
  
```  
IMAGEREL expression  
```  
  
## <a name="remarks"></a>설명  
 결과 값은 RVA 또는 상대 가상 주소 라고도 합니다.
