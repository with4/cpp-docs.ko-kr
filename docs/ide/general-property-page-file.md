# <a name="general-property-page-file"></a>일반 속성 페이지(파일)

**솔루션 탐색기**에서 파일을 선택하면 **구성 속성** 노드 아래의 **일반** 속성 페이지에 다음과 같은 속성이 포함됩니다.

**빌드에서 제외**  
파일을 현재 구성에 맞게 빌드해야 하는지 여부를 지정합니다.

프로그래밍 방식으로 이 속성에 액세스하려면 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.ExcludedFromBuild%2A>을 참조하세요.

**도구**  
이 파일을 빌드하는 데 사용할 도구입니다. 자세한 내용은 [사용자 지정 빌드 도구 지정](../ide/specifying-custom-build-tools.md)을 참조하세요.

프로그래밍 방식으로 이 속성에 액세스하려면 <xref:Microsoft.VisualStudio.VCProjectEngine.VCFileConfiguration.Tool%2A>을 참조하세요.

**구성 속성** 노드 아래의 **일반** 속성 페이지에 액세스하는 방법에 대한 자세한 내용은 [프로젝트 속성 작업](../ide/working-with-project-properties.md)을 참조하세요.

Windows 이외 프로젝트의 경우 [Linux C++ 속성 페이지 참조](../linux/prop-pages-linux.md)<!-- or [C++ Cross Platform Property Page Reference](../linux/prop-pages-linux.md)-->를 참조하세요.
