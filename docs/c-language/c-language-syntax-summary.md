# <a name="c-language-syntax-summary"></a>C 언어 구문 요약
이 섹션에서는 C 언어 및 Microsoft 전용 C 언어 기능에 대한 자세한 설명을 제공합니다. 이 섹션의 구문 표기법을 사용하여 언어 구성 요소에 대한 정확한 구문을 확인할 수 있습니다. 구문에 대한 설명은 이 설명서에서 항목 설명 섹션에 나와 있습니다.
