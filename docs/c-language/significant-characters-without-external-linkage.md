# <a name="significant-characters-without-external-linkage"></a>외부 링크가 없는 중요한 문자
**ANSI 3.1.2** 외부 링크가 없는 유효 문자 수  
  
 식별자는 247자에 상당합니다. 컴파일러는 식별자에 사용할 수 있는 문자 수를 제한하지 않지만 제한을 초과하는 문자는 모두 무시합니다.
